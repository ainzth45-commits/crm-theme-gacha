# 🎲 Theme Gacha — สุ่มส่วนลดธีม

เว็บเล็กๆ (static) สำหรับให้พนักงานกดสุ่มส่วนลดเอาไปแลกซื้อธีม ใช้กับ **iPad Air 5** เป็นหลัก · ขึ้น GitHub Pages ได้เลย (ไม่ต้อง build)

## วิธีใช้
1. กรอกรหัสพนักงาน → เข้าสู่ระบบ (ไม่มี OTP)
2. กดปุ่ม **สุ่มส่วนลด** (ได้ **วันละ 1 ครั้ง** ต่อรหัส)
3. ได้ส่วนลด → เก็บไว้ในเครื่อง · กด **ใช้ส่วนลด** เมื่อจะเอาไปแลกธีมจริง (กดแล้วส่วนลดหาย)
4. **เปิด Catalog** ดูธีม + ราคาหลังลด
5. **เปลี่ยนรหัส** (มุมขวาบน) = ออกไปหน้า login

## กติกา
- ธีมราคา **10 เหรียญ** ทุกธีม (เหรียญจริงนอกระบบ)
- สุ่มได้วันละครั้ง · สุ่มใหม่จะทับส่วนลดเดิม
- จำส่วนลด/วันที่สุ่มล่าสุด ใน `localStorage` ต่อรหัสพนักงาน

ค่าเฉลี่ย ≈ 48% · ปรับได้ที่ `js/gacha.js` (ตัวแปร `RATE`)

## เพิ่มรูปธีม
วางไฟล์รูปที่ `assets/themes/` แล้วอ้างใน `data/themes.js` (`img`) — ถ้ายังไม่มีรูปจะโชว์ป้ายสีแทนอัตโนมัติ

## โครงไฟล์
```
index.html · css/style.css · js/{app,gacha,sound,catalog}.js · data/themes.js · assets/themes/
```
ทุกอย่างเป็น vanilla HTML/CSS/JS · เสียงสังเคราะห์ด้วย Web Audio (ไม่มีไฟล์เสียง)
